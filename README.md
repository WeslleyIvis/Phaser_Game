## Phaser 3
    * Pointer Events 
        EX: event.on("pointerover")
            pointerover - hovering
            pointerout - not hovering
            pointerup - click and release
            pointerdown - just click

    * Gameobject events
        EX: obj.on("animationstart")
            animationstart
            animationrepeat
            animationupdate
            animationcomplete
